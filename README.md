#  Tempus — Banco de Tempo

Requisitos do desenvolvimento de um banco de tempo, feito para facilitar a aquisição de experiência dos estudantes da UnB, a troca de atividades entre eles e possibilitar a troca de serviços sem custo financeiro.

O funcionamento consiste em um sistema de troca de créditos em  que cada crédito equivale a 1 hora e pode ser usado para pedir a realização de atividades e cada novo usuário entra no sistema já com alguma quantidade gratuita de créditos para poderem participar antes do primeiro serviço.

Os requisitos podem ser acessados de forma organizada via este link: [Tempus](https://si-2026-1.github.io/Tempus/)

## Contexto acadêmico

Projeto realizado com o intuito de ser uma experiência prática na produção de requisitos do desenvolvimento de um sistema.

Ele ocorreu no contexto da matéria de sistemas de informação da turma 2 no semestre 2026.1 da UnB.
